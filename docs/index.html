How to read this site

This corpus is every .apk in Alpine main + community / x86_64, unpacked and indexed by llvm-readobj-20 --elf-output-style=JSON. Two parallel static-analysis passes feed the site:

  • Survey (scripts/survey.py) — fast in-Python parser of the JSON dumps. Counts every ELF field, cross-references against the spec, classifies each file into one of five categories (loadable / debuginfo / relocatable / cross_arch / other) and emits structured anomalies (e.g. ph_load_wx when a PT_LOAD is W+X).
  • eu-elflint oracle (scripts/elflint_run.py) — wraps elfutils 0.190 eu-elflint -q over the same corpus, normalises each finding into a message template, and buckets by template. Independent second opinion: we agree on what's broken when both fire.

Use the nav to drill into any axis. The anomalies and elflint pages are the two strongest signals about non-standard ELFs in this corpus.
